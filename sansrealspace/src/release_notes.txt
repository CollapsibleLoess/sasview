Release Notes
=============

SANS Real-Space Simulation version 0.3

Package name: sans.sansrealspace

1- What's New in Version 0.3

	- Release date: 
	   
	- Simulation of oriented systems.
	- Now possible to add an object using the 'addObject' method rather
	  than using the VolumeCanvas as a factory.
	- Added validation and tests for oriented shapes.

2- Downloading and Installing

	2.1- System Requirements:
		- Python version >= 2.4 should be running on the system

	2.2- Installing:
		- Install realSpaceModeling by running setup_noadpt.py
		- Install sansrealspace by running setup.py
